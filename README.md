# Digit Recognition using MNIST

## Project Overview
This is a Machine Learning project that recognizes handwritten digits (from 0 to 9) using the famous **MNIST dataset**. The model analyzes images of handwritten numbers and accurately predicts which digit it is.

## Domain
* Machine Learning (Internship Project)

## Features
* Loads and processes the MNIST handwritten digits dataset.
* Trains a Machine Learning model to recognize numbers.
* Predicts outputs with high accuracy.

## Technologies Used
* **Python**: Core programming language.
* **Scikit-Learn**: For building the training model.
* **Matplotlib & NumPy**: For data visualization and array handling.
* **VS Code**: The development environment.
